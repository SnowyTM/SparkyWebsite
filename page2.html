Details About Sparky


Sparky is a Discord Bot and was made to handle a multiple variety of commands. The bot was made using 100% Python and uses multiple different libraries that Python has to offer.

Sparky has plenty of modules within her files, the biggest being the gacha module with a boasting 2.3mb worth of storage and about 700 lines of Python code!

Sparky was originally planned to be hooked up with a Transformer making her a potential AI, but that project was quickly put to hold and instead will have a different bot do that.

Sparky being a disocrd bot, has many uses besides being a fun and entertaining bot to use! Sparky can also do moderation commands such as Kick, Ban, Unban, and Purge!

The bot contains around 2,000 to 3,000 lines of Python code.

Sparky has nine(9) modules excluding the main code file.

Sparky utilizes a special weight RNG system when calculating for the chances of getting a title!

Sparky actually has two creators, the other one not wanting to be mentioned since she said she "didn't do much."


A little fun fact, Sparky actually has the tools needed to become a Transformer, the only issue was that Snowy can't get the algorithm to run on his laptop!

Sparky's Commands


SparkyHelp Command

This is the SparkyHelp command! To initiate it, you type: /sparkyhelp

This command gives you the list of commands and other features available in Sparky, simply click either the two buttons bellow to navigate through the pages!


Math Command

This is the Math Command! To initiate it you type: /math

This command will give you the sum of two numbers that you've provided.
Sparky can only do addition, subtraction, multiplication, and division. No modulus sadly.

Member Count Command

This is the Member Count Command! To initiate it, you type: /membercount

This command shows you the amount of members in your server/guild excluding (Not including) bots/applications such as Dyno, Sparky, Carl-Bot, and much more.

Donate Command

This is the Donate Command! To initiate it, you type: /donate

This command is supposed to give you a paypal link to my account or my guild leader's account (since she is struggling financially) but as of now it has no links yet.
Also, Paypal requires me to be 18+ so only my guild leader's paypal will be available if you'd like to support her and her family.

SlotMachine Commnad

This is the SlotMachine Command! To initiate it, you type: /slotmachine

This is just a simple slotmachine, not much effort was put into it so please do forgive me if it's really hard.
Just press the pull button and it'll start the slot machine, be quick though before the embed updates!

BlackJack Command

This is the BlackJack Command! To initiate it, you type: /blackjack

This is a simple blackjack game, hit or stand against a machine dealer!

Gacha Command

This is the Gacha Command! To initiate it, you type: /gacha

This is one of the more complicated commands, it gives you a panel/embed and a few buttons.
One(1) pull or anything that ends with a "pull" means that you are collecting a title.
New titles will have a star next to them(Only works up to 100 pulls).
Rarities are accurate as Sparky is using a weight rng system.

Titles Command

This is the Titles Command! To initiate it, you type: /titles

This command gives you a list of titles and their droprate chances.
Click the buttons below the embed to navigate through the pages.

Show Inventory and Show Title Command

This is two commands! The ShowInventory Command and the ShowTitle Command!
To initiate ShowInventory, you type: /showinventory
To initiate ShowTitle, you type: /showtitle and then the title you want to show off!

ShowInventory just shows the titles you have stored/collected so far. Use the buttons to navigate through your inventory! ShowTitle shows the title you want to show! Some titles may or may not have their own special theme.


Moderation Commmands Below

Kick Command

This is the Kick Command! To initate it, you need to be an admin and type:
/kick + @username of the person you want to kick and then optional reason.

It's pretty self explantory.

Ban Command

This is the Ban Command! To initiate it, you need to be an admin and type:
/ban + @username of the person you want to ban and then optional reason.

Keep in mind that banning someone using Sparky is a permanent ban unless you unban them through Sparky, or other bots, or manually navigate through the server settings.

Unban Command

This is the Unban Command! To initiate it, you need to be an admin and type:
/unban + USER_ID of the person you want to unban.

To collect the user's ID, go to their profile and you'll see a BBQ kind of icon, click it and find "Copy User ID" or something like that.

Purge Command

Watch it all PURGE!

This is the Purge Command! To initiate it, you need to be an admin and type:
/purge and then then the amount you want to get rid, and optionally target a specific user only as well.

Fair warning, purge limit can only reach up to 100 messages as the Discord API refuses bots to go beyond that. There is solutions around it but it is not worth the complexity putting on a singular command.

Track Member Joins Command

This is the Track Member Joins Command! To enable it, you need to be an admin and type:
/track_member_joins and the channel you want the welcome messages to go to.

Track Member Joins command pretty much acts as a tracker to whoever joins the server. Sparky will greet them in the channel specified during the calling of the command, you can switch where Sparky can send messages by simply calling the command again and assigning it to a new channel. This command will also soon be able to track member leaves so keep an eye out for that update!
